# README

## Init

```bash
npm install -g @vue/cli

# update the package
npm update -g @vue/cli

# check application is configured
vue --version
```

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint

# fix the code
npm run lint --fix
```
